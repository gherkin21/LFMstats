\doxysubsection{Detailed Description}
The main application window for the Last.\+fm Scrobble Analyzer. 

This class manages the user interface, coordinates interactions between the UI elements, the \doxylink{class_last_fm_manager}{Last\+Fm\+Manager} (for fetching data), the \doxylink{class_database_manager}{Database\+Manager} (for storing/loading data), the \doxylink{class_settings_manager}{Settings\+Manager} (for config), and the \doxylink{class_analytics_engine}{Analytics\+Engine} (for processing data). It displays various statistics and charts based on the loaded scrobble data. UI responsiveness is maintained by performing database loading and data analysis in background threads. @inherits QMain\+Window

Detailed Description

Worker object performing actual Last.fm API requests in a separate thread.

Encapsulates the QNetworkAccessManager and handles the request construction, sending, and basic reply processing (parsing JSON, extracting data/errors). Runs within the thread managed by LastFmManager. @inherits QObject

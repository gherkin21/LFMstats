Detailed Description

Manages the persistence of scrobble data to the local disk.

Provides asynchronous methods for saving fetched scrobble pages and loading stored scrobbles. Data is organized into weekly JSON files per user. Uses QtConcurrent for background tasks. @inherits QObject

Asynchronously queues a list of scrobbles for saving to the database.

The save operation is performed in a background thread. Empty usernames or scrobble lists are handled gracefully.

Parameters
pageNumberThe page number corresponding to this batch of scrobbles (used for signalling).
usernameThe Last.fm username the scrobbles belong to. Cannot be empty.
scrobblesThe list of ScrobbleData items to save.

Detailed Description

Manages interaction with the Last.fm API for fetching scrobble data.

Handles asynchronous fetching of recent tracks, pagination, rate limiting (via delays), and error handling including retries for specific errors. Uses a LastFmWorker object running in a separate thread for network requests. @inherits QObject
